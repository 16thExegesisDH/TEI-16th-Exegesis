# TEI-16th-exegesis

This repository contains the XML-TEI texts of the corpus of the project *16th Century Exegesis of Paul*.  
The data are identical in the following repository: [link to-app-data/corpus]().

Repository description:
- The list of texts with further details can be found in the [CSV table](https://github.com/16thExegesisDH/TEI-16th-Exegesis/blob/main/TEI_table.csv) of the repository.
- The "schema" folder contains the project ODD in different formats (.odd, .html, .rng).
- The "data" folder contains the texts in XML-TEI format.

Description of the file encoding:
to come

To understand the progress status of the texts (*gold*, *silver*, *bronze*, or *to come* quality), simply consult the [CSV table](https://github.com/16thExegesisDH/TEI-16th-Exegesis/blob/main/TEI_table.csv) of the repository and/or the teiHeader of the XML-TEI files.

# identifier

*CR : corpus 
* A; B; C : type of corpus
* _suffixe : M_id of RRP database, manifestation id = precise text in a precise copy of a Work.
